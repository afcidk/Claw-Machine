# 已佔用 pin 角
* RA0, RA1: 搖桿X,Y
* RB0: 紅外線
* RB1: 七段顯示器用的微動開關
* RC6, RC7: 七段顯示器的 CLK/DIO
* RD0 ~ RD5: 馬達

# 安裝提示
* RB1 微動開關： C 接正，另一個用 NO。記得要接**下拉電阻**
* 七段顯示器： DIO 接 pin 26, CLK 接 pin 25
* 搖桿：X 接 RA0, Y 接 RA1, B 不用管
* 搖桿馬達輸出： RD0~RD5

----
# Buzzer
* 需要兩個 pin 角接到 RD0, RD1
* 0 0->沒聲音
* 0 1->正常
* 1 0->失敗
* 1 1->成功
**一旦輸入都是 0 0 音樂會馬上停止，其他的會等到音樂播完再換**
